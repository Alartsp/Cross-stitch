# Cross Stitch v3 Exact Template Overlay

Ця версія працює вже під реальний шаблон:
- окремо завантажуєш фото;
- окремо завантажуєш шаблон (саме той PNG/JPG/скрін, який потрібен);
- app шукає центри овальних комірок по темному контуру;
- кольори накладаються точно в знайдені комірки поверх оригінального шаблону.

## Як апдейтнути GitHub Pages
1. Замінити файли в root repo.
2. Відкрити:
   `https://alartsp.github.io/Cross-stitch/?v=3exact`
3. Якщо старий PWA був встановлений — видалити ярлик і відкрити сайт заново.

## Рекомендації
- для шаблону краще використовувати чистий скрін/PNG без зайвих кнопок;
- якщо комірки знаходяться погано — підкрути `Поріг темних контурів` і `Мін./Макс. площа елемента`;
- для простих фото тримай 8–12 кольорів як стартовий режим.

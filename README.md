# Cross Stitch v3.4 Object Mask Fix

v3.4 фіксить кейси типу щітки на складному/небілому фоні.

## Суть фіксу
- оцінюється фон по border-pixels;
- будується маска самого об'єкта за віддаленням від фону + насиченістю;
- використовується найбільша foreground-область;
- у сітку потрапляють тільки комірки, що лежать усередині маски об'єкта.

# Cross Stitch Oval Template v2

Ця версія переводить фото у вигляд, близький до наданого зразка:
- овальні комірки;
- зсув кожного другого ряду;
- біле поле з відступами;
- контур овалів + кольорове заповнення всередині.

## Як оновити GitHub Pages
1. Замінити файли в root repo.
2. Відкрити сайт з cache-bypass:
   `https://alartsp.github.io/Cross-stitch/?v=2template`
3. Якщо був старий PWA — видалити ярлик і відкрити заново.

## Що ще можна зробити в v3
- завантаження реального PNG-шаблону як фону;
- точніше підганяння під конкретний шаблон/друк;
- PDF експорт;
- ручний crop / zoom.

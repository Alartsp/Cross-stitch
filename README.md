# Cross Stitch v3.1 Exact Template Overlay

v3.1 додає коректне збереження пропорцій фото:
- фото вписується у bbox знайдених комірок без розтягування;
- кольори беруться по реальних центрах комірок;
- якщо співвідношення сторін фото і шаблону відрізняється, app додає поля замість деформації.

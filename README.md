# Cross Stitch v3.2 Full Crosses Only

v3.2 додає важливу умову: **ніяких неповних хрестиків / неповних комірок**.

## Що змінилось
- фото вписується в цілу кількість комірок;
- краї більше не обрізають часткові стібки;
- або комірка заповнюється повністю, або лишається порожня.

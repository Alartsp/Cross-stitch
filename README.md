# Cross Stitch v4 Manual Fit

Це свідомий pivot від невдалої «магічної автоматики» до керованого режиму.

## Що робити
1. Завантажити фото.
2. Завантажити шаблон.
3. Натиснути `Розпізнати шаблон`.
4. Підкрутити:
   - `Масштаб об'єкта (%)`
   - `Зсув X / Y`
   - `Режим фону`
   - `Поріг відділення від фону`
5. Натиснути `Згенерувати`.

## Коли це краще
- вузькі предмети / щітки / packshot;
- складний або неідеально білий фон;
- коли треба передбачуваний результат, а не випадкова автоматика.
